###title: 分析预测接口

- 使用方式
    - 通过将要处理的数据放到`GET`的body里面以json的格式发送过来

- 接收数据格式

```json
{
"code": 200,
"msg":"找到对应结果",
"data": {
"ticker": "000001",
"category": 5,"finanace_data":[
{
"date": "2023-03-31"","value": "14469000000"},
{
"date" : "2022-12-31","value": "45407000000"},
{...},
...
]
}
}
```

- 数据说明
    - 'code':表示http请求的状态码(int)
    - 'msg':附加信息(string)
    - 'data':数据内容(dictionary)
    - 'ticker':股票编号(string)
    - 'category':数据类型(ing)
    - 'date':股票某一时间点(string)
    - 'value':对应数据类型的值(int)

- 结果返回
    - 返回一个json形式的数据

- 返回数据格式
```json
{
    "ticker": "0000001",
    "category": 1,
    "score": 87.123415
}
```

- 数据说明
    - 'ticker':股票编号
    - 'category':数据类型
    - 'score':评分
