---
title: 状态码
---

| 名称               | 数值 | 描述                |
| ------------------ | ---- | ------------------- |
| OK                 | 200  | 响应成功            |
| UNKNOWN_ERROR      | 500  | 未知错误            |
| NO_SUCH_FIELD      | 4101 | 没有搜索字段        |
| INDEX_CREATE_ERROR | 4201 | 新增索引失败        |
| INDEX_NOT_FIND     | 4202 | 索引库中无对应结果  |
| MYSQL_CREATE_ERROR | 4301 | Mysql数据库创建失败 |
| MYSQL_SEARCH_ERROR | 4302 | Mysql数据库访问失败 |
