# 项目简介

为帮助广大A股投资者高效获取A股各家上市公司财务信息，辅助他们进⾏投资决策，我们开发了上市公司财务信息收集与分析系统。本系统的数据模块使用分布式爬虫实时收集信息，分析预测模块使用机器学习模型预测股价，可视化模块使用多种表格和统计图让用户直观了解信息并发现数据之间深层次关联。基于这三大模块我们实现了行情查询、数据可视化、个人理财、量化交易等多种功能。
